## 🚀 Quick start

**Start developing.**

Navigate into your new site’s directory and start it up.

```shell
cd my-default-starter/
yarn add
```

Start in development mode

```shell
yarn develop
```

Clean cache

```shell
yarn clean
```

Build the app for production

```shell
yarn build
```
