# Leandro Arturi's Portfolio

### React, Strapi, Node and Mongo

## 🚀 Quick start

**Start developing.**

Navigate into your new site’s directory and start it up.

```shell
yarn install
```

Start in development mode

```shell
yarn start
```

Clean cache

```shell
yarn clean
```

Build the app for production

```shell
yarn build
```
